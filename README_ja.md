# My Antigravity Skills Collection 🚀
これは、私が個人的に作成・メンテナンスしている [Antigravity](https://github.com/GoogleDeepMind/antigravity) (または Claude Code などの Agent ターミナル) 専用のローカルスキル (Skills) コレクションです。
これらのカスタムスキルを作成することで、AI アシスタントに特定領域の専門的な能力を持たせ、通常の対話の限界を突破し、より高度な自動化ワークフローを実現できます。

## 🛠️ 収録スキル一覧
### 1. WeChat 公式アカウント記事取得ツール (wechat-article-summarizer)
**概要**:
WeChat 公式アカウントの強力なスクレイピング対策や「環境異常」による CAPTCHA ブロックを回避するために特化して開発された、自動取得スキルです。AI に WeChat の記事を要約させる際、デフォルトのブラウザノードや通常の HTTP リクエストでは CAPTCHA が発動して失敗することが多いため、これを作成しました。
**主な機能**:
- **シームレスな CAPTCHA 回避**: Python スクリプトを使用して標準的なブラウザのリクエストヘッダーを偽装し、ヘッドレスで HTML ソースコードを直接取得。CAPTCHA ポップアップを完全に回避します。
- **高精度な抽出**: 正規表現に基づき、不要な CSS、JS、HTML タグを高精度で除去し、記事のタイトル、著者、および本文のみを抽出します。
- **AI との完全自動連携**: ユーザーが `mp.weixin.qq.com` のリンクを送信したことを Agent が検知すると、シームレスにこのスクリプトを呼び出してプレーンテキストを取得し、高品質な構造化された要約を出力します。
**利用シーン**:
「この記事を要約して：https://mp.weixin.qq.com/s/...」

### 2. Web ページからクリーンな PDF への変換ツール (webpage-to-pdf)
**概要**:
広告、サイドバー、ポップアップなどを含む複雑な Web ページを、電子書籍のようにクリーンな PDF ファイルに変換することに特化したローカルスキルです。
**主な機能**:
- **遅延読み込み (Lazy Loading) の回避**: Playwright を使用して Chromium ブラウザを操作し、人間の自然なスクロール動作をシミュレートすることで、ページ内のすべての「遅延読み込み」画像が完全に読み込まれてからキャプチャを実行します。
- **DOM のスマートな浄化**: 特定の CSS ルールを自動的に注入し、ナビゲーションバー、サイドバー、フローティング広告、フッターの著作権情報などの不要な要素を強制的に非表示にします。
- **レイアウトのリセット**: Web ページ本文の幅制限を自動的に解除 (100% フル幅) し、エクスポートされる A4 サイズの PDF をコンパクトで高解像度かつ美しく仕上げます。
**利用シーン**:
「この Web ページをクリーンな PDF として保存して：https://www...」
