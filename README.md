# My Antigravity Skills Collection 🚀
这里是我个人制作并维护的 [Antigravity](https://github.com/GoogleDeepMind/antigravity) (或 Claude Code 等 Agent 终端) 专属本地技能（Skills）汇总合集。
通过编写这些自定义技能，我可以让 AI 助手掌握更多垂直领域的专属能力，突破常规交互的限制，实现更深度的自动化工作流。

## 🛠️ 包含的技能列表
### 1. 微信公众号文章抓取助手 (wechat-article-summarizer)
**简介**：
针对微信公众号极强的反爬虫和“环境异常”人机验证拦截机制，专门开发的一个免验证自动化抓取技能。当遇到需要 AI 总结微信文章时，默认的浏览器子节点或普通的 HTTP 请求常常会触发人机验证导致抓取失败。
**核心功能**：
- **无感绕过验证**：底层通过 Python 脚本及伪装标准浏览器请求头，直接无头获取公众号 HTML 源码，完美绕开“环境异常”的验证弹窗。
- **自动精准提取**：基于正则高精度剥离多余的 CSS、JS 和 HTML 标签，仅保留最核心的文章标题、作者和正文。
- **AI 全自动衔接**：Agent 只要检测到用户发送了 `mp.weixin.qq.com` 的链接，便会无缝调用此脚本获取纯文本，最后自动输出高质量的文章结构化总结。
**使用场景**：
“帮我总结一下这篇文章：https://mp.weixin.qq.com/s/...”
### 2. 网页转纯净版 PDF 助手 (webpage-to-pdf)
**简介**：
一个专门用于将包含广告、侧边栏和弹窗的复杂网页转换成排版干净、如同电子书般纯净的 PDF 文件的本地技能。
**核心功能**：
- **突破懒加载**：使用 Playwright 底层驱动 Chromium 浏览器，模拟真实人类用户的自然下拉滚动行为，确保网页内所有的“懒加载”图片完全加载完毕后再进行抓取。
- **DOM 智能净化**：自动注入特定的 CSS 规则，暴力屏蔽常见的导航栏、侧边栏、悬浮广告、底部版权信息等无关干扰元素。
- **重置排版**：自动释放网页正文的宽度限制（100% 满宽），使最终导出的 A4 尺寸 PDF 版面紧凑、高清且美观。
**使用场景**：
“帮我把这个网页干净地存为PDF：https://www...”

### 3. 多语种官方社媒推文助手 (official-social-posts)
**简介**：
一个用于日常官方账号内容生产的多语种社媒写作技能，默认面向 Twitter/X，也可用于其他官方社媒。它会先读取上下文、项目资料和已有官推风格，判断是否已经是连续工作流；如果不是首次运行，就默认沿用上一次确认过的语种、长度、语气、输出格式和配图偏好。
**核心功能**：
- **上下文优先**：首次执行时先回看对话、项目文件和品牌素材；已有需求时自动继承上次配置，减少重复确认。
- **多语种成套输出**：按需求输出中文、日文、英文等多语种推文，每个语种都配套 hashtag 和图片方案。
- **选题与热点结合**：可根据用户指定主题写作，也可结合当前热点提出多个选题供选择。
- **官方账号语气控制**：强调克制、自然、不过度营销，适合品牌官号发帖和回帖。
- **配图流程**：优先使用项目里的 logo、poster、品牌素材；含文字的社媒图优先用 HTML/CSS 等可控方式生成，减少多语种错字。
**使用场景**：
“基于当前 AI 热点，帮我写一条日英双语官推，并配 hashtag 和宣传图。”
