# Output Patterns

## Standalone Multilingual Post

```text
**Japanese**

[ready-to-post copy]

Hashtags:
#[brand] #[topic] #[category]

Image:
[path or visual concept]

**English**

[ready-to-post copy]

Hashtags:
#[brand] #[topic] #[category]

Image:
[path or visual concept]
```

## Reply To Another Post

```text
[acknowledge the original point]

[one practical observation]

[subtle brand relevance, if appropriate]
```

## Topic Proposal

```text
1. [Topic] - [why it fits the brand and current discussion]
2. [Topic] - [why it fits]
3. [Topic] - [why it fits]

Recommended: [topic], because [short reason].
```

## Hashtag Guidelines

Use 3-5 hashtags. Mix one brand tag, one category tag, and 1-3 topic tags.

Examples:

```text
#ApiMax #AIAPI #LLM #従量課金 #コスト最適化
#ApiMax #AIInfrastructure #LLMAPI #PayAsYouGo
```
